6.4. Actions View

The properties view allow annotations to be specified against actions and their parameters.

The default shortcut for the view is alt+shift+Q,5. On the left-hand slide of the slider the annotations for the action itself can be maintained:

The annotations that can be created by the view on actions are:

  • @Named(value:String)

    Corresponds to the Named column

  • @MemberOrder(sequence:String, name:String)

    Corresponds to the MO#sequence and MO#name columns. The format of the sequence is a Dewey-decimal string "x.x.x".

  • @Disabled(value:When)

    Corresponds to the Disabled column

  • @Executed(value:Executed.Where)

    Corresponds to the Executed column

  • @Hidden(value:When)

    Corresponds to the Hidden column

  • @Debug

    Corresponds to the Debug column

  • @Exploration

    Corresponds to the Exploration column

  • @DescribedAs(value:String)

    Corresponds to the DescribedAs column

On the right-hand side of the slider the annotations for the action parameters can be maintained:

The annotations that can be created by the view on actions parameter are broadly those applicable for properties and string properties. In the same way that the string properties view disables rows for non-string properties, so are individual cells disabled when they do not apply to specific action parameters:

  • @Named(value:String)

    Corresponds to the Named column

  • @Optional

    Corresponds to the Optional column

  • @TypicalLength(value:int)

    Corresponds to the TypicalLength column

  • @MaxLength(value:int)

    Corresponds to the MaxLength column. Only applicable for string parameters.

  • @MultiLine(numberOfLines:int,preventWrapping:boolean)

    Corresponds to the ML#numberOfLines and ML#preventWrapping columns. Only applicable for string parameters.

  • @RegEx(validation:String,format:String,caseSensitive:boolean)

    Corresponds to the RE#validation, RE#format and RE#caseSensitive columns. Only applicable for string parameters.

  • @DescribedAs(value:String)

    Corresponds to the DescribedAs column.
