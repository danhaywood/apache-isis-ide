5. Browsing the Code

Most coding conventions tend to specify that either instance variables should (lexically) appear before methods in the class definition. Or maybe they state that methods come first, and then methods. For domain objects written against the Naked Objects programming model, however, we've found that it makes more sense to group class members by responsibility.
