4. Templates

The NO IDE has two sets of templates, usually invoked using ctrl+space. These can be viewed using Windows>Preferences...>Java>Code Style>Code Templates:

As the screenshot indicates, it is also possible to associate individual key bindings to these templates (though the NO IDE doesn't do this by default).

The domain object templates fall into two categories:

  • those that apply within a type definition (but not in a method definition), and

  • those that apply before the type definition.

The service templates support the implementation of repositories and factories.

In all cases the templates come with detailed tooltips making it easy to understand what they will do before actually applying them.
