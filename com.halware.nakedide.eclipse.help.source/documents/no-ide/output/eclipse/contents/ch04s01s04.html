1.4. Icon Project Wizard

The icon project wizard is used to create an icon project that holds a set of icons for use in Naked Objects applications. You are free to use these icons in any application that runs on an NOA-compliant framework, but in no other.

To invoke the wizard, use: File>New...>Project...>Naked Objects>Icon Project:

Since this wizard always produces the same set of icons, the project name is defaulted for you.
