4.1. Launch the Drag-n-Drop Client

Open up the DnD Client launch, and then press Run. The DnD client should be shown:

Each of the icons corresponds to the services in the application project, with the menu shown corresponding to the actions of com.mycompany.myapp.services.CustomerRepository . In the nakedobjects.properties file these are registered as services and so are added to the desktop.
