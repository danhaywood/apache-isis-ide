6. Change an Action Signature

Occasionally it is necessary to change an action signature, either to add or remove parameters, or perhaps change the order. For actions the Naked Objects programming model requires that an action's supporting methods must have the same parameter types as the action that they support.

Although the NO IDE can rename supporting methods, it does not (yet) refactor the parameters of supporting action methods. Therefore you must change each individually. That said, if you make a mistake and forget to rename one of them, the project builder should spot this.
