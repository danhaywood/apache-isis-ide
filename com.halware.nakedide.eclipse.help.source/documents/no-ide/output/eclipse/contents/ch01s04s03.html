4.3. Run the tests

On the launch configuration dialog and under the JUnit type is a launch configuration for CustomerTest:

Select this and Run it. In Eclipse the JUnit view should open up, with the tests having passed:

If you refresh the Package Explorer view (File > Refresh) a new 'tmp' directory should have been created. This contains a transcript of the tests.

You have now created the sample application, launched it as either the DnD or web clients, and run the sample tests. Let's now have a closer look at the code.
