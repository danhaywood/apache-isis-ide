6.2. String Properties View

The string properties view allow annotations specific to strings to be specified against properties with a return type of java.lang.String. All properties are shown (so that switching between this view and the properties view doesn't jar) but those properties that are not string properties are disabled.

The default shortcut for the view is alt+shift+Q,2:

The annotations that can be created by the view are:

  • @MaxLength(value:int)

    Corresponds to the MaxLength column

  • @MultiLine(numberOfLines:int,preventWrapping:boolean)

    Corresponds to the ML#numberOfLines and ML#preventWrapping columns

  • @RegEx(validation:String,format:String,caseSensitive:boolean)

    Corresponds to the RE#validation, RE#format and RE#caseSensitive columns
