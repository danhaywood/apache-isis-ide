6. Add a New Property

The NO IDE comes with a set of templates that make it easy to add new properties, collections and actions to existing objects. Let's use these now to add a new middleInitial property to the Customer class.

First, then, open up this class in the editor, and position the cursor within the type definition, after the FirstName folded region. Then press ctrl+space 3 times to bring up the Domain Object templates. Then scroll down to the nopc template and hit enter:

In the Customer class will be the skeleton for a new property. As the template's tooltip indicates, the most commonly used methods are provided. Enter the following values for the placeholders:

  • PropertyName = MiddleInitial

  • propertyName = middleInitial

  • PropertyType = String

When done, hit escape; the following should have been added to your class:

If everything is correct then the code should compile - check Eclipse's Problems view to make sure.
