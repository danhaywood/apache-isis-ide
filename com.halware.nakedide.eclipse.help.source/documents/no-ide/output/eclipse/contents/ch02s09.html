9. Relationship between the NO IDE and the NOF Runtime

The NO IDE is a development environment whose primary aim is to make it easy to develop NOA-compliant applications. A secondary goal is to make it easy to run those applications on the NOF, as a reference implementation.

However, the NO IDE does not provide any functionality for deploying NOA applications into a system test or production environment. Under the NOF Maven is used to do this; consult the NOF documentation for details.
