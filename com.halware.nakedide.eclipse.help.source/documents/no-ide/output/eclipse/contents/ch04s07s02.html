7.2. Rename Method

The rename method refactoring renames all methods for an property, collection or action, plus the instance variable if appropriate (for property or collection).

Select the method, and use Refactor>Rename...:

Any of the methods - supporting or otherwise - can be renamed; the prefix should remain the same. For example, here the Customer#validateCustomerNumber is being renamed to validateCustomerId:

This will have the effect or renaming the customerNumber property.
