9.2. Naked Objects Nature

The Naked Objects nature should be applied to application projects (containing the domain objects and supporting services). The NO IDE refactorings & project builder only run when this nature is enabled:

The application project wizard will automatically install the nature, but it can also be added or removed manually:

One time when this is useful if you want to refactor some code which doesn't strictly conform to the Naked Objects programming model, and need to temporarily disable refactoring. Alternatively you might do this if you have uncovered a bug with the NO IDE's refactorings or builder which is preventing you from working (in which case, please log the defect).
