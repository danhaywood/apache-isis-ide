1.3. Client Project Wizard

The client project wizard is used to create a client project that holds configuration data and launch configurations. These allow a domain application to be launched using the Naked Objects Framework.

To invoke the wizard, use: File>New...>Project...>Naked Objects>Client Project:

As for the application and fixture project wizards, this wizard is based in large part on Eclipse's own plug-in project wizard. However the first page (shown above) is new, requiring the developer to enter the application project containing the domain objects and the fixture project that has its fixtures. The application project name is used to derive the name of the client project itself, and both projects are used in the classpath of the project. The classpath of the client project depends fully on the Naked Objects Framework runtime.

In addition to setting up the skeleton of the project, the wizard also creates 2 launch configurations to run up either the Drag-n-Drop client or the Web local server client.
