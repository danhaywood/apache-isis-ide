6.3. Collections View

The properties view allow annotations to be specified against collections.

The default shortcut for the view is alt+shift+Q,4:

The annotations that can be created by the view are:

  • @Named(value:String)

    Corresponds to the Named column

  • @MemberOrder(sequence:String, name:String)

    Corresponds to the MO#sequence and MO#name columns. The format of sequence is a Dewey-decimal string "x.x.x".

  • @TypeOf(value:String)

    Corresponds to the TypeOf column. The value of

  • @Disabled(value:When)

    Corresponds to the Disabled column

  • @NotPersisted

    Corresponds to the NotPersisted column

  • @Hidden(value:When)

    Corresponds to the Hidden column

  • @DescribedAs(value:String)

    Corresponds to the DescribedAs column
